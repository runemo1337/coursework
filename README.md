# 📄 Инструкция для проекта "IT-Профориентация" [Ruslan]

🚀 _Как запустить_ [Ruslan]

- Склонируй репозиторий.
- В терминале напиши: `npm install` (установит все детали).
- Запуск проекта: **`npm run dev`** (не `npm start`!).
- Открой ссылку в браузере (обычно `localhost:5173`).

📂 _Где мы работаем (Папка src)_ [Ruslan]

- **components/** — Твоя основная зона. Создавай здесь визуальные части (окна, кнопки).
- **assets/** — Сюда закидывай картинки и иконки.
- **types/** — Моя зона. Здесь я описываю правила данных.
- **data/** — Моя зона. Здесь лежат сценарии и тексты квестов.
