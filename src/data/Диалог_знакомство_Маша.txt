ВОПРОС 1: Что можешь рассказать про фронтенд?
Игрок: Что ты можешь рассказать про фронтенд?

Маша: Нууууу... если вкратце, то фронтенд — это то, что видит пользователь. Мы делаем кнопки, окошки, картинки, анимации — всё, с чем ты взаимодействуешь на сайте. Когда ты что-то вводишь или нажимаешь — фронтенд собирает это и отправляет на сервер. А когда сервер отвечает — мы показываем эти данные красиво и понятно. Ну и ещё мы делаем так, чтобы на разных устройствах и в разных браузерах всё отображалось одинаково круто. Здесь есть где разгуляться, если ты творческая личность.


ВОПРОС 2: На каких языках вы пишете?
Игрок: На каких языках вы пишете?

Маша: Весь фронтенд держится на трёх китах — HTML, CSS и... барабанная дробь... JavaScript! Мой любимый язык, потому что с его помощью ты можешь написать что угодно — любой эффект и любую штуку. Сделать так, чтобы сайт был интересным. И ограничено там всё только твоим воображением!

Вариант 2.1: Расскажи про HTML
Игрок: Расскажи про HTML.

Маша: HTML — это каркас страницы. Он говорит браузеру: «вот здесь заголовок, здесь абзац, здесь картинка». Без HTML для браузера всё — просто сплошной текст. А с HTML у тебя появляется структура.

Вариант 2.2: Расскажи про CSS
Игрок: Расскажи про CSS.

Маша: Ну CSS — это про стиль. Я серьёзно. CSS — это каскадные таблицы стилей. Он определяет внешний вид. Если HTML создаёт структуру страницы, то CSS отвечает за то, как это будет выглядеть — цвета, размеры, расположение, отступы.

Вариант 2.3: Расскажи про JavaScript
Игрок: Расскажи про JavaScript.

Маша: Ну, если коротко: с помощью HTML разработчик собирает «скелет». Потом работает с CSS-кодом для стилизации страницы — покрывает «кожей». А в финале запускает «кровеносную, пищеварительную и дыхательную системы», программируя на JavaScript. То есть JavaScript делает сайт не просто красивым, но ещё и живым! С его помощью сегодня мы имеем такие блага, как всплывающие окна, анимация, кнопки лайков и формы для отправки информации. Это и многое другое написано на JS. Так что если пойдёшь на фронтенд — не прогадаешь, уж поверь мне.

Вариант 2.4: Понятно
Игрок: Понятно.
